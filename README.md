# AI Chronicles

> Working notes on building with AI coding agents. Principles and patterns over feature lists.

An evergreen, growing collection of chronicles drawn from primary sources and practitioner experience. Each entry distils a topic into the principles and patterns that have held up across model generations, deliberately kept generic so they age well as specific tools, version numbers, and feature names shift underneath them.

New chronicles are added as the field moves; existing ones are revised when the underlying ideas change.

## Chronicles

- [001 — Effective Claude Code Setup](chronicles/001-effective-claude-code-setup.md) — Mental model, context budgeting, `CLAUDE.md`, skills, plugins, tools/MCP, scaling from one agent to many, and workflow patterns.

## Contributing

Pull requests welcome for principles that have held up across multiple model generations, or patterns with concrete adoption stories behind them. Speculation, marketing claims, and untested wisdom belong in [GitHub Discussions](../../discussions), not the main chronicles.
